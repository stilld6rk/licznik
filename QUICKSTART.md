# ⚡ Quick Start - TL;DR

## 1️⃣ Przygotuj GitHub repo
```bash
git init
git add .
git commit -m "init"
git push -u origin main
```

## 2️⃣ Wejdź na https://railway.app
- Zaloguj się GitHub
- "Create New Project"
- Wybierz swoje repo

## 3️⃣ Dodaj PostgreSQL
- W projekcie kliknij "Add Plugin"
- Wybierz "PostgreSQL"
- Railway zrobi resztę

## 4️⃣ Ustaw zmienne (Variables w Railway)

Skopiuj z twojego kodu:

```
DISCORD_BOT_TOKEN=paste_twój_token
GUILD_ID=1473620113021472870
ROLE_ID=1512190222320799865
WEBHOOK_URL=paste_twój_webhook
HARD_LOGIN=stillindark
HARD_PASSWORD=RIPn#W5.Moa:2FnR
HARD_PIN=99711
LIMIT=4
HEADLESS=true
```

## 5️⃣ Czekaj 2-3 minuty
Railway deployuje automatycznie i bot powinien być online!
